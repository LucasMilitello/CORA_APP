3) MAPEAMENTO DIRETO TK -> PYSIDE6

3.1 Bootstrap e ciclo de vida
- Tk:
  - root = tk.Tk(), root.mainloop(), protocol("WM_DELETE_WINDOW", ...)
- Qt:
  - QApplication, QMainWindow, app.exec(), closeEvent(self, event)
- Arquivo alvo:
  - qt/cora_interface_qt.py

3.2 Variaveis de estado
- Tk:
  - StringVar/DoubleVar/BooleanVar
- Qt:
  - atributos Python + atualizacao explicita de widgets
  - sinais/slots para sincronizar UI
- Dica:
  - dominio em atributos; widget apenas reflete e emite eventos

3.3 Layout principal (duas telas)
- Tk:
  - pagina configuracao + pagina visualizacao (pack_forget/pack)
- Qt:
  - QStackedWidget para trocar pagina
  - QSplitter para painel lateral + canvas
  - QTabWidget para resultados/guia/progresso

3.4 Campos e botoes principais
- Tk:
  - Entry, Button, Combobox, Checkbutton
- Qt:
  - QLineEdit, QPushButton, QComboBox, QCheckBox
- Dialogos:
  - filedialog.askdirectory -> QFileDialog.getExistingDirectory
  - messagebox.* -> QMessageBox.*
  - colorchooser -> QColorDialog.getColor

3.5 Revisao de agrupamento (parte critica)
- Tk:
  - Treeview editavel + selecao + preview + confirmar/cancelar
- Qt recomendado:
  - QDialog + QTableView + QAbstractTableModel
  - coluna "Incluir" com check (Qt.CheckStateRole)
  - delegate para tempo (0h/24h/48h)
  - QDialogButtonBox para Confirmar/Cancelar/Iniciar processamento
- Regras de negocio reutilizaveis:
  - collect_grouping_entries(...)
  - build_groups_from_entries(...)
  em qt/core/tk_portable_pack.py

3.6 Processamento em background
- Tk:
  - threading.Thread + queue.Queue + root.after(90, poll)
- Qt:
  - manter threading.Thread + queue.Queue + cancel_event
  - trocar root.after por QTimer periodico
- Reuso direto:
  - services/processing_service.run_processing_worker

3.7 Progresso e cancelamento
- Tk:
  - popup + barra + ETA + botao cancelar
- Qt:
  - QDialog modeless custom (ou QProgressDialog)
  - QProgressBar + QLabel
  - botao cancelar -> setar cancel_event

3.8 Viewer de resultados e metricas
- Tk:
  - Text + FigureCanvasTkAgg
- Qt:
  - QPlainTextEdit (read-only)
  - FigureCanvasQTAgg para matplotlib
- Preview e metricas por grupo:
  - reaproveitar logica de montagem de texto e overlays

3.9 Configuracao de contornos
